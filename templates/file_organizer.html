File Organizer Script

Organize your files into folders based on their type.

About the Script

This Python script helps you organize files in a specified folder by automatically moving them into subfolders based on their file extensions. For example, it moves .mp4 files into a "Videos" folder and .mp3 files into an "Audios" folder.

Features

  • File Type Detection: Recognizes .mp4 and .mp3 files for organization.
  • Folder Creation: Automatically creates "Videos" and "Audios" folders if they donâ€™t already exist.
  • Customizable Path: Allows users to specify the folder they want to organize.
  • Error Handling: Skips unsupported file types and folders.

How to Use

  1. Download the script using the button below.
  2. Ensure Python is installed on your system.
  3. Edit the main_folder variable in the script to the path of the folder you want to organize.
  4. Run the script using python app.py in your terminal or command prompt.
