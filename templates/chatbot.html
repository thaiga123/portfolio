Python Chatbot Project

An interactive chatbot built using Python and JSON

About the Chatbot

This chatbot is a simple yet powerful Python program that interacts with users, providing answers to queries, performing basic math operations, conducting quizzes, and more.

Features

  • Friendly Greetings: Responds to common greetings like "hi" or "hello."
  • Math Solver: Performs basic math calculations, such as addition, subtraction, multiplication, and division.
  • Quizzes: Asks random quiz questions and evaluates your answers.
  • Time Updates: Tells you the current time when asked.
  • Error Handling: Politely handles inputs it doesn't understand.

Technologies Used

  • Python
  • JSON (for storing data)

How to Use

  1. Download the chatbot's files using the button below.
  2. Ensure Python is installed on your system.
  3. Extract the ZIP file and navigate to its directory.
  4. Run the script using python app.py in your terminal.
  5. Start interacting with the chatbot by typing your queries.
